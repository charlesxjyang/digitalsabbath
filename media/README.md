# Digital Sabbath — Media Assets

Brand assets, social media templates, and app store materials.

## Structure

```
media/
├── brand/        # Logo files, color palette, fonts doc
├── flyers/       # Instagram posts, stories
└── screenshots/  # App store screenshots
```

## Brand Guidelines

- **Primary color:** Golden amber (#F4A623)
- **Background:** Warm cream (#FFF8F0) to soft gold (#F9D89C)
- **Text:** Dark warm brown (#4A3728)
- **Font:** Georgia or similar serif
- **Tone:** Warm, calm, inviting. Never preachy or scolding.
