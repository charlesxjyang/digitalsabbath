# Digital Sabbath — Backend API

**Status: Planned**

Simple API to power the participation counter on the landing page.

## Planned Features

- Endpoint to register a participant (anonymous, no account needed)
- Endpoint to get the current participant count
- Lightweight — this doesn't need to be complex

## Planned Stack

- Cloudflare Workers or simple Node.js service
- KV store or SQLite for the counter
